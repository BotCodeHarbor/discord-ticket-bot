# Discord Ticket Bot

A professional Discord ticket system built with discord.py.

## Features

- Slash command ticket panel
- Ticket creation button
- Private ticket channels
- Ticket close button
- Modern Discord embeds
- Easy to customize

## Installation

### Clone the repository

```bash
git clone https://github.com/BotCodeHarbor/discord-ticket-bot.git
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Configure bot token

Edit:

```python
TOKEN = "YOUR_BOT_TOKEN"
```

### Run

```bash
python bot.py
```
